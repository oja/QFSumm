This directory contains the collected aspect-oriented annotations. 
Each file contains one aspect for a domain as indicated in its name:
- eq_100_geo: Earthquake, Geography
- eq_100_rescue: Earthquake, Recovery
- fraud_100_penalty: Fraud, Nature
- fraud_100_nature: Fraud, Penalty

These collectively make up the AspectNews dataset.
Each .csv file contains a comma-seperated annotations in JSON format. 
The corresponding .txt files are human-readable versions of the .csv files.

- summaries.json contains the SPACE dataset
